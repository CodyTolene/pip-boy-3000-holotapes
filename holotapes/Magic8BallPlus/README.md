###Description ###

The classic Magic 8 Ball and all 20 of its responses. Spin/click either wheel to get a new response!

### Controls ###

| Input                | Action                                                                |
| -------------------- | --------------------------------------------------------------------- |
| Left wheel (rotate)  | Shuffles the ball for a new response                                  |
