# Signal Match

## Info

**Author:**

- [@Radiomaru7](https://github.com/Radiomaru7)

## Description

A Simon Says-style memory game for the Pip-Boy 3000. Watch the sequence of
numbered panels light up, then repeat it back correctly to advance.

### Controls

- **Hold Left Wheel** to start a new game (or retry after a loss).
- **Turn either wheel** to select a value during input.
- **Tap Left Wheel** to confirm your selection.

### How to Play

1. Watch the pattern as it plays on screen ("WATCH THE PATTERN").
2. Repeat the sequence back by turning either wheel and tapping the left wheel
   to confirm each step ("TURN EITHER WHEEL, TAP LEFT TO CONFIRM").
3. A correct match advances you to the next round.
4. An incorrect match ends the game and shows your score. Hold the left wheel to
   retry.
