# iPip Media Player v1.3.0

### Modifications by Joe Byers

### Based on iPip Media Player v1.2.1 by Cody Tolene

iPip Media Player turns folders inside the Pip-Boy's `MUSIC` directory into browseable radio stations and playlists.

This version builds upon Cody Tolene's original iPip Media Player and adds a variety of usability, playback, sorting, and interface improvements developed during real-world testing on the Wand Company Pip-Boy 3000.

## Major Additions

* Upload-order playlist support
* Upload / A-Z / Z-A song sorting
* Sequential playback enabled by default
* Long song title scrolling
* Improved wheel responsiveness
* Increased filename/path-length support
* Improved playlist and playback synchronization
* Expanded interface audio feedback
* Enhanced random playback behavior

## Included Files

* `app.js` — Working application build
* `app_beautified.js` — Human-readable source for review and development
* `metadata.json` — App metadata
* `assets/iPip.png` — Application icon
